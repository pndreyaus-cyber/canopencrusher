Эта ветка содержит код для тестирования библиотеки CANopenNode.

CANopenNode - универсальная библиотека, подходящая под разные микроконтроллеры. 
Чтобы она подходила под STM32 Arduino необходимо изменить ряд файлов.
Какие файлы и как написано в инструкции к библиотеке:
https://github.com/CANopenNode/CANopenNode/tree/ef9ac3a2279e34855a20c787fc1bc48bc995ec22/example -- эти файлы надо изменить и  еще добавить стандартные библиотековские файлы

Возможности этой папки
- отправлять SDO с индексом 6060
